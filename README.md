# 📊 Samsung Supply Chain & Logistics Dashboard (Power BI)

## 🚀 Project Overview

This project is a Power BI dashboard designed to analyze and visualize Samsung’s supply chain and logistics performance. It provides insights into key metrics such as inventory, transportation, delivery performance, and overall supply chain efficiency.

---

## 🎯 Objectives

* Monitor supply chain performance
* Identify delays and inefficiencies
* Analyze logistics and delivery trends
* Support data-driven decision making

---

## 📌 Key Features

* 📦 Inventory Analysis
* 🚚 Logistics & Transportation Insights
* ⏱ Delivery Performance Tracking
* 📈 KPI Dashboard with interactive visuals
* 🔍 Drill-down analysis for deeper insights

---

## 🛠 Tools & Technologies Used

* Power BI
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

---

## 📂 Files in Repository

* `SAMSUNG Supply Chain & Logistics Dashboard.pbix` → Main Power BI file

## ▶️ How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore the interactive dashboard
